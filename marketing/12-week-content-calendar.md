## PRODUCTION PLAN

### File naming convention
`canvas/ig-{pillar_code}-{week}-{number}.html`
- P1 = `ig-t1-w1-1.html` (Training Intelligence, Week 1, Post 1)
- P2 = `ig-m1-w1-1.html` (Mistake Correction)
- P3 = `ig-p1-w1-1.html` (Product Proof)
- P4 = `ig-r1-w1-1.html` (Race Strategy)

### Weekly workflow
1. **Sunday**: Write copy for upcoming 3–4 posts
2. **Monday**: Build HTML canvases, push to GitHub
3. **Mon–Sat**: Capture screenshots via `?slide=N`, approve, publish
4. **Sun**: Review metrics from previous week, identify top performers
5. **Mon (ramp period)**: Configure Meta ads to boost top post from previous week
